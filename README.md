# Bharat Gold Market Intelligence Volatility Dashboard

## Project Overview

This project analyzes historical gold prices in India by integrating multiple macroeconomic datasets, including international gold prices, USD/INR exchange rates, import duty, GST, RBI repo rate, and annual gold imports.

## Objective

To build an interactive Tableau dashboard that analyzes gold price trends, policy impacts, market volatility, and economic factors influencing domestic gold prices.

## Tools Used

- Tableau
- Microsoft Excel
- Git & GitHub

## Project Structure

- data/raw – Original datasets
- data/processed – Cleaned master dataset
- dashboard – Tableau workbook
- docs – Project documentation
- images – Dashboard screenshots
- notebooks – Analysis notebooks
- sql – SQL scripts
